# Fuel

Fuel is a protein-forward food logger PWA designed for fast daily use.

The goal is not to become another giant nutrition-sludge app.
It should help with:
- logging meals quickly
- seeing daily calories and protein at a glance
- staying aware of intake without spreadsheet punishment

## Current shape
- **Today** — totals, quick add, meals logged
- **Log** — manual meal entry
- **History** — lightweight pattern view

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```
